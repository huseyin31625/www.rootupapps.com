How to force BSOD on Boot

Firstly, you need to write these commands:

  dism /mount-image /imagefile:<YOUR install.wim FILE> /mountdir:<YOUR MOUNT DIRECTORY> /index:<1|2|3|4>
  Dism /Add-Driver /Image:<YOUR MOUNT DIRECTORY> /Driver:<INF FILE OF YOUR BSOD ON BOOT DRIVER> /forceunsigned
  dism /unmount-image /mountdir:<YOUR MOUNT DIRECTORY> /commit

Open UltraISO.

Open your Windows installation ISO file.

Delete install.esd from ISO contents.

Place the install.wim file.

Save ISO file.

Open VMWare Workstation.

Create a virtual machine with your ISO.

And install the Windows.
